# Carlux Inventory Grid

Live Site: [carlux-phi.vercel.app](https://carlux-phi.vercel.app/)

A responsive vehicle inventory dashboard built with Next.js and Tailwind CSS, featuring real-time search, price sorting, and optimized component architecture.

---
## Features
- Next.js & TypeScript for fast, scalable, and type-safe web development.
- Tailwind CSS for responsive and modern UI design.
- Fetches vehicle data from API
- Displays image, title, brand, and price
- Fully responsive layout
- Real-Time Search
- Sorting Functionality
- Loading State
- Error Handling
- Empty State Handling
- Performance Considerations

---

##  Tech Stack

- **Framework:** Next.js  (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel
- **Data Source:** DummyJSON API
